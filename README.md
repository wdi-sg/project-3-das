# Commonplaces
* https://commonplaces-app.herokuapp.com/

## Ruby Version and System dependencies
* Rails
* Devise
* Cloudinary
* Carrierwave

## Configuration
* Git clone the repository
* Run `bundle install` on your terminals to install all dependencies
* Run `rails db:migrate`
